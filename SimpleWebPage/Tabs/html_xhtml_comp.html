HTML y XHTML no son lo mismo

¿Qué diferencias hay entre HTML y XHTML?

A pesar de que sus nombres se parecen, HTML y XHTML tienen entre sí varias diferencias significativas. Para empezar, XHTML es una versión de HTML combinada con XML, por lo que puede tener reglas más estrictas. Para poder entender esto a más detalle, realizamos una comparación entre 20 de las características de ambos lenguajes. A continuación, la comparación sobre las principales características de HTML5 contra XHTML 1.1 y XHTML 2 respectivamente:
